= 2026-05-29 — Bundle-only distribution; npm retired

== Summary

The release `ui-bundle.zip` is the only distribution channel. npm publish was removed from CI; consumer docs point at xref:guide:npm-deprecation.adoc[npm deprecation] for migration and lessons learned.

== CI

* `.github/workflows/release.yml` — no `pnpm pack`, no `antora-dark-theme-*.tgz` on GitHub Releases, no `npm publish`, no `id-token` / `environment: npm`.

== Documentation

* `npm-deprecation.adoc` — status, migration, lessons learned, archived former install commands.
* `README.md`, `README.adoc`, `installation.adoc`, `index.adoc`, `configuration.adoc` — bundle-only; link to deprecation page instead of inline npm sections.
* `code-blocks.adoc` demo listing — `antora` CLI only, not `antora-dark-theme` npm install.

== package.json

* `"private": true` — prevents accidental registry publish; Antora demo build still uses `pnpm` in this repo.

== Registry

Maintainers deprecate or unpublish `antora-dark-theme` on npmjs separately. Catalog entry belongs on Antora Supplemental docs / extensions registry with the bundle URL.
