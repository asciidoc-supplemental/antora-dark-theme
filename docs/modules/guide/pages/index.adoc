= Getting Started
:navtitle: Getting Started
:description: Quick introduction to the Antora Dark Theme supplemental UI.

Antora Dark Theme adds dark mode to https://antora.org[Antora] documentation sites.

== Install habit

Pin the GitHub release `ui-bundle.zip` in your playbook—see xref:installation.adoc[Installation].

Catalog: link:https://antora-supplemental.github.io/docs/[Antora Supplemental docs portal].

If you previously used npm: xref:npm-deprecation.adoc[npm deprecation].

== Features

* Dark mode toggle, system preference, `localStorage` persistence, no FOUC
* Release bundle includes Default UI + theme (no UI fork)
* Navbar, nav, content, code, tables, admonitions, footer styling
