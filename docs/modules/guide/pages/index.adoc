= Getting Started
:navtitle: Getting Started
:description: Quick introduction to the Antora Dark Theme supplemental UI.

Antora Dark Theme adds dark mode to your https://antora.org[Antora] documentation sites.

== The habit you should adopt

**Use the GitHub release `ui-bundle.zip`.** That is the better solution—not npm, not sync scripts from `node_modules`, not stacked supplemental directories.

[source,yaml]
----
ui:
  bundle:
    url: https://github.com/antora-supplemental/antora-dark-theme/releases/download/v1.0.9/ui-bundle.zip
    snapshot: true
----

Catalog and discovery: link:https://antora-supplemental.github.io/docs/[Antora Supplemental docs portal].

The https://www.npmjs.com/package/antora-dark-theme[npm package] is deprecated and will be unpublished. xref:npm-deprecation.adoc[Migrate off npm] if you still depend on it.

== Features

* *Dark mode toggle* - Sun/moon control in the navbar
* *System preference* - Follows OS dark mode when you have not chosen a fixed theme
* *Persistent preference* - Stored in `localStorage`
* *No FOUC* - Dark class applied before paint
* *Non-invasive* - Release bundle includes Default UI + theme; no UI fork required
* *Broad styling* - Navbar, nav panels, content, code, tables, admonitions, footer
